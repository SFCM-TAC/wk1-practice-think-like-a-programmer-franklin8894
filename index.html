Think Like a Programmer

Pretend you are writing a plugin that automatically harmonizes with MIDI input. What instructions would you give a computer to do this programmatically?
Assign notes (frequencies) to be played simultaneously with the note that is triggered in MIDI. For example, assign an E to a C, so that a major 3rd harmony would sound if the C is triggered.
Pick an instrument and pretend you are writing an app that mimics that instrument. How would you model the istrument in a way the computer would understand?
